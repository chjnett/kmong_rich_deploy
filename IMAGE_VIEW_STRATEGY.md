# 상품 상세 페이지 이미지 뷰어 개선 전략 (Image View Strategy)

## 1. 개요 (Overview)
현재 상품 상세 페이지에서 사용 중인 슬라이드 방식의 이미지 뷰어는 고정된 높이와 `object-cover` 속성으로 인해 다양한 비율의 상품 사진이 상하좌우로 잘리는(Cropping) 현상이 발생하고 있습니다. 이를 해결하고 명품 브랜드 사이트와 같은 프리미엄 스크롤 경험을 제공하기 위해 이미지 레이아웃을 전면 개편합니다.

## 2. 문제점 분석 (Current Issues)
- **이미지 크롭핑 (Cropping):** `object-cover` 속성은 부모 컨테이너를 가득 채우기 위해 이미지의 일부를 잘라내어 시각적 정보 손실을 발생시킵니다.
- **수평 슬라이드의 한계:** 슬라이더 방식은 한 번에 하나의 이미지만 볼 수 있으며, 상품의 전체적인 디테일을 연속적으로 확인하기 어렵습니다.
- **모바일 경험:** 엄지손가락으로 가로로 터치하여 미는 동작보다 아래로 자연스럽게 스크롤하는 동작이 사용자에게 더 직관적입니다.

## 3. 개선 방향 (Proposed Solutions)

### A. 슬라이더에서 수직 리스트(Vertical List)로 전환
- 모든 이미지를 한꺼번에 아래로 나열하여 사용자가 스크롤만으로 모든 사진을 감상할 수 있게 합니다.
- **데스크탑:** 좌측 이미지 리스트는 스크롤되고, 우측 상품 정보(이름, 가격, 버튼)는 화면에 고정(Sticky)되어 항상 보이도록 구성합니다.
- **모바일:** 수평 슬라이더 대신 수직으로 길게 이미지를 배치하여 자연스러운 탐색 환경을 제공합니다.

### B. 비율 기반 레이아웃 (Aspect Ratio Optimization)
- `object-cover` 대신 `w-full h-auto`를 사용하여 이미지 본연의 비율을 최대한 보존합니다.
- 배경색을 적절히 배치하여 (예: `#f9f9f9` 또는 `muted/20`) 다양한 사이즈의 이미지가 이질감 없이 어우러지도록 합니다.

### C. 인터랙티브 요소 강화 (Interactions)
- **스크롤 애니메이션:** 이미지가 화면에 나타날 때 부드러운 Fade-In 효과를 적용하여 고급스러운 느낌을 줍니다.
- **줌 인 효과:** 이미지를 클릭하거나 호버 시 디테일을 더 자세히 볼 수 있는 옵션을 고려합니다.

## 4. 기대 효과 (Expected Outcome)
- **정보 전달 극대화:** 잘리는 부분 없이 상품의 모든 면을 온전하게 보여줄 수 있습니다.
- **프리미엄 무드:** 명품이나 고가 브랜드에서 주로 사용하는 '스크롤 갤러리' 방식을 채택하여 웹사이트의 전반적인 품질을 높입니다.
- **체류 시간 증가:** 사용자가 스크롤을 통해 상품을 더 깊이 있게 탐색하도록 유도합니다.
