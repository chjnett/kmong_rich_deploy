## 2. 메인 페이지 - 상단 히어로(Hero) 섹션
기존의 단순 전체화면 배너에서 벗어나, 트렌디한 화면 분할(Split Screen) 레이아웃을 도입합니다.

*   **레이아웃**: 화면을 좌/우 2개의 큰 영역으로 분할.
*   **왼쪽 영역 (공지사항 / 텍스트 롤링)**:
    *   **내용**: 사용자에게 띄우는 주요 **공지사항** 혹은 이벤트 안내.
    *   **관리**: 관리자(Admin) 페이지의 공지사항 DB와 연동하여, Admin에서 수정 및 추가 시 실시간으로 왼쪽 화면에 반영되게 함.
    *   **UI**: 텍스트가 부드럽게 페이드인 되거나 슬라이드업 되는 등 깔끔하고 동적인 모션 적용.,이미지가 여러개면 자동으로 슬라이드 되게
*   **오른쪽 영역 (메인 비주얼)**:
    *   **내용**: 요청하신 지정 로고 이미지 적용.
    *   **경로**: `/Users/cheonhyeonjun/workspace_kmong/01kmong_eterna/main_logo.jpeg`
    *   화면을 꽉 채우되 비율이 깨지지 않게 끔 우측 영역을 커버(`object-cover`)하도록 스타일링.

## 3. 메인 페이지 - 상품 진열 섹션 (Best Seller / New Arrivals)
첨부된 두 번째 레퍼런스 이미지 스타일의 비대칭 수평 스크롤(Horizontal Scroll) 또는 캐러셀(Carousel)을 구현합니다.

*   **레이아웃 구성**:
    *   **좌측 고정 요소 (30% 비율)**: 
        *   섹션 타이틀 (예: `BEST SELLER`)
        *   "지금 가장 인기있는 상품을 만나보세요" 같은 간결한 서브 텍스트.
        *   전체보기(`Shop All ↗`) 버튼 스티키(Sticky)하게 위치.
        *   이거를 admin페이지에서 설정 가능하게
    *   **우측 스크롤 영역 (70% 비율)**:
        *   여유 공간부터 마우스 휠이나 스와이프로 넘겨볼 수 있는 가로형 상품 카드 리스트 배치.
        *   **데이터 연동**: 기존 상품 DB를 그대로 불러와 노출. (이미지, 이름, 가격 등)
*   **상품 카드 디자인**:
    *   상품 이미지를 크고 시원하게 배치.
    *   하단 또는 이미지 위에 심플한 텍스트로 상품명과 폰트 크기/색상 대비를 활용한 할인율/가격 표시.

## 4. 상품 상세 정보 페이지 (Product Detail Page)
메인 페이지의 미니멀/갤러리 컨셉을 맥락 끊김 없이 상세 페이지까지 이어갑니다.

*   **이미지 갤러리 (좌측 혹은 상단 큼직하게)**:
    *   상품 이미지가 돋보이도록 복잡한 테두리를 없애고 여백(Margin)을 매우 넓게 활용.
*   **제품 정보 및 구매 영역 (우측 또는 하단)**:
    *   상품설명, 옵션(사이즈/색상) 선택, 장바구니/구매 버튼.
    *   수평선(Divider)을 최소화하고 폰트의 크기와 두께(Weight) 차이만으로 정보의 위계질서 표현.
    *   기존 DB에 있는 제품 상세 HTML/텍스트나 옵션 데이터 구조를 그대로 사용하여 뷰(View)만 교체.
